This project provides Pascal bindings for the SANE library (libsane).  
SANE itself is distributed under the GNU General Public License (GPL) version 2 or later.  
It is a standardized API for accessing image scanners and similar devices under Unix-like operating systems.  
For details, see the original SANE project: https://sane-project.gitlab.io/sane-project/
