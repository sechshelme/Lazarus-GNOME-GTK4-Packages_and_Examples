This project provides Pascal bindings for the GNU MPFR library.
MPFR itself is distributed under the GNU Lesser General Public License (LGPL) version 3 or later.
It is a well-established library for multiple-precision floating-point computations with correct rounding.
For details, see the original MPFR project: https://www.mpfr.org/
