This project provides Pascal bindings for the GNU MPC library.  
MPC itself is distributed under the GNU Lesser General Public License (LGPL) version 3 or later.  
It is a well-established library for multiple-precision complex floating-point computations with correct rounding.  
For details, see the original MPC project: https://www.multiprecision.org/mpc/
