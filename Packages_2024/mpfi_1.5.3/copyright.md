Pascal bindings for MPFI  
Released under the MIT License  

This project provides Pascal bindings for the MPFI library.  
MPFI (Multiple Precision Floating-Point Interval library) is based on MPFR and GMP, providing arbitrary-precision interval arithmetic.  
MPFI itself is distributed under the GNU Lesser General Public License (LGPL).  
For more information, visit the original MPFI project page:  
https://gforge.inria.fr/projects/mpfi/  
or the mirror repository at GitHub:  
https://github.com/sergiud/mpfi
